Example gallery

Example gallery#

Welcome to the abTEM gallery. These notebooks demonstrate specific examples of many of the concepts learned in the Walkthrough, which you should start with. These examples are currently not very actively maintained.

  • basic Examples that cover the basics image simulation, these examples are generally appropriate for new users. Many of these examples may also be used as a template for performing a specific common type of simulation.

  • specialized Examples that introduce a less common topic in simulation of electron microscopy. Some of these examples assume that the user have experience abTEM.

If youd like to add your book to this list, simply add an entry to this gallery.yml file and open a Pull Request to add it. For more detailed instructions see our guide on contributing to abTEM.
